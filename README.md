# TropeSearch

• Developed a movie discovery web application enabling dynamic filtering across 5+ attributes including tropes,
genres, ratings, year, and popularity.
• Implemented state management using React Hooks to synchronize UI behavior across multiple browsing work-
flows (solo and group discovery modes).
• Integrated relational film and trope data using Supabase (PostgreSQL) with external movie metadata to deliver
responsive search results and structured content views.
